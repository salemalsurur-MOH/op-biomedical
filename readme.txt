الخيار الأسرع — Netlify Drop  (دقيقتان فقط)
═══════════════════════════════════════════════════════
1) افتح:  https://app.netlify.com/drop
2) اسحب هذا المجلد كاملاً وأفلته في الصفحة
3) ينشأ لك رابط مباشرة، انسخه وشاركه

ملاحظة: لو سجّلت دخول قبل السحب، الرابط يبقى دائماً.
بدون تسجيل دخول، الرابط مؤقت لساعة.

═══════════════════════════════════════════════════════
الخيار الثاني — GitHub Pages  (مجاني ودائم)
═══════════════════════════════════════════════════════
1) أنشئ حساب على:  https://github.com
2) من الزر الأخضر New repository:
     - Repository name: medical-maintenance  (أو أي اسم)
     - علّم Public
     - اضغط Create repository
3) في الصفحة التالية اضغط uploading an existing file
4) اسحب الملفات الثلاثة + مجلد assets، ثم Commit changes
5) من تبويب Settings → Pages
     - Source: Deploy from a branch
     - Branch: main / root → Save
6) انتظر دقيقة، يطلع لك رابط:
     https://USERNAME.github.io/medical-maintenance

ملاحظات مهمة
═══════════════════════════════════════════════════════
• البيانات (الملفات اللي تضيفها، التثبيتات، المجموعات)
  محفوظة في متصفح كل مستخدم على حدة (localStorage).

  وهذي خطوة منفصلة.

  بصيغة JSON، وزر «استيراد» لاسترجاعها على جهاز آخر.
