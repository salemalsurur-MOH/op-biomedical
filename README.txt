═══════════════════════════════════════════════════════
المزامنة بين الأجهزة — جاهزة تلقائياً
═══════════════════════════════════════════════════════
الموقع متصل بـ Firebase (مشروع op-biomedical). أي ملف
تضيفه أو تثبّته أو مجموعة تنشئها من أي جهاز يظهر على
بقية الأجهزة فوراً. ستلاحظ شارة "متزامن" خضراء أعلى
الصفحة عند الاتصال.

═══════════════════════════════════════════════════════
الخيار الأسرع — Netlify Drop  (دقيقتان، رابط دائم)
═══════════════════════════════════════════════════════
1) سجّل حساب مجاني (مرة واحدة):
       https://app.netlify.com/signup
   استخدم Google أو إيميلك المعتاد.

2) افتح بعد تسجيل الدخول:
       https://app.netlify.com/drop

3) اسحب هذا المجلد كاملاً (publish) وأفلته في الصفحة.

4) ينشأ لك رابط فوري على الشكل:
       https://اسم-عشوائي.netlify.app
   من Site settings ← Change site name تقدر تغيّره مثلاً:
       https://op-biomedical.netlify.app

5) احفظ الرابط — ادخله من أي كمبيوتر بدون رفع أي ملف
   بعد اليوم.

ملاحظة مهمة: لما تحدّث الموقع لاحقاً، اسحب المجلد
نفسه على نفس الموقع (Deploys ← Drag and drop)، الرابط
ما يتغير، تتحدث الملفات فقط.

═══════════════════════════════════════════════════════
الخيار الثاني — GitHub Pages  (مجاني ودائم)
═══════════════════════════════════════════════════════
1) أنشئ حساب على:  https://github.com
2) من الزر الأخضر New repository:
     - Repository name: medical-maintenance
     - علّم Public  ← ثم Create repository
3) اضغط uploading an existing file
4) اسحب الملفات الأربعة + مجلد assets، ثم Commit changes
5) من Settings ← Pages:
     - Source: Deploy from a branch
     - Branch: main / root  ← Save
6) ينشأ لك رابط دائم:
     https://USERNAME.github.io/medical-maintenance

═══════════════════════════════════════════════════════
الخيار الثالث — Firebase Hosting  (لو تبي على نفس الدومين)
═══════════════════════════════════════════════════════
بما أن قاعدة البيانات على Firebase أصلاً، تقدر تنشره
على نفس المشروع وتحصل رابط:
     https://op-biomedical.web.app
يحتاج تثبيت Firebase CLI من جهازك مرة وحدة:
     npm install -g firebase-tools
     firebase login
     firebase init hosting   (اختر مشروع op-biomedical،
                               مجلد publish، Single-page: لا)
     firebase deploy
