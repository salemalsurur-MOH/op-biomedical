## 📊 الجداول المدرجة

### جدول تحليل الزمن (Durations Analysis)
- من تاريخ العطل إلى تاريخ البدء
- من تاريخ البدء إلى تاريخ التوقيع
- الفترة الكلية من العطل إلى التوقيع

### جدول الموظفين والأداء
- عدد الأوامر المسندة
- متوسط زمن الإصلاح
- نسبة الإنجاز لكل موظف

### جدول الأجهزة المتكررة
- عدد مرات الأعطال
- متوسط زمن الإصلاح
- التصنيف (أولويات)

### جدول تفاصيل الأوامر
- رقم التحكم
- الجهاز والصانع
- تاريخ العطل والحالة
- المسؤول والقسم

## 🛠️ التقنيات المستخدمة

- **Frontend:** React / Next.js 14
- **Charts:** Chart.js + React ChartJS 2
- **Data Processing:** xlsx library
- **Styling:** CSS3 + RTL support
- **State Management:** React Hooks

## 💾 كيفية استخدام الملف

1. **رفع ملف Excel:**
   - اضغط على زر "رفع ملف Excel جديد"
   - اختر ملف بنفس البنية الأصلية
   - سيتم معالجة البيانات تلقائياً

2. **الفلترة والتصفية:**
   - استخدم القوائم المنسدلة لاختيار المنشأة
   - حدد نطاق التاريخ المطلوب
   - اختر نوع أمر العمل

3. **قراءة البيانات:**
   - المؤشرات في الأعلى توضح الملخص العام
   - الرسوم البيانية توضح التوزيعات
   - الجداول توضح التفاصيل
